Description

Ansible is an automation tools, you can use it to run automation on remote machine but it does’t support run on Windows, it only supports run on Linux machine. There is a very important feature, It doesn’t require the target machine to install ansible agent, it only requres you install ansible on the managment machine

Install

Ansible depends on python, so if you want to install ansible, you must first install python, it support python2.7 and python3, here is the document how to install Ansible.
